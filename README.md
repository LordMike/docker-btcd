# btcd server

btcd server started with command line arguments matching environment variables.


### Environment variables
* BTCD_TESTNET
* BTCD_PROXY
* BTCD_PROXYUSER
* BTCD_PROXYPASS
* BTCD_NOONION
* BTCD_ONION
* BTCD_ONIONUSER
* BTCD_ONIONPASS
* BTCD_TORISOLATION
* BTCD_UPNP
* BTCD_EXTERNALIP
* BTCD_NORPC
* BTCD_NODNSSEED
